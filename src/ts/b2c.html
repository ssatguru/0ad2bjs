About
I wrote this application to help Blender users develop animations for CloudParty's default avatars.
It fixes an issue with the Blender Collada Import/Export plugin.The plugin alters the bone roll of the skeleton on import/export and because of this any animations created, using this skeleton, does not play correctly on the original CloudParty skeleton.
This application corrects the animation by "undoing" the bone roll introduced by the plugin.
You do not need this application if you are developing animations for avatars other than the default CloudParty avatar.

See
CloudParty Forum Post - Fix for the Blender Avatar Animation Issue
or
CloudParty Forum Post - Possible fix for avatar animations in Blender
for some discussions on this issue.

This application has been tested in Chrome 24.0.1312.57 m and FireFox 18.0.2 on Windows 7
Quick Instructions
  1. Create animation in Blender using the blender files from the "Downloads" section below.
    • The blender files containe the CP Default avatar imported into Blender.The avatar has been oriented towards you (the negative Y direction) for ease of creating animations.
      There is no need to rotate the avatar by 180 to face positive Y which is the CloudParty's forward direction. The converter takes care of this.
    • Do not add any bindpose_skeleton. The converter adds the necessary bindpose_skeleton.
  2. Export animation in Collada format
    • You only need to export the b_root skeleton. The mesh is not required.
    • Make sure the following Collada options are selected
      • Selection only
      • Include Armatures
      • Export for Second Life

  3. Use the Converter to load this file and convert it to CloudParty format.
    When the conversion is done, a link will appear. Right click the link and choose "Save the link as.." to save the file as a .daeanim file. (The browser generates the file in memory and gives a random, unique name to it. Currently, this seems to be the best way to save it.)
    The conversion happens in your browser, in your computer. Nothing is sent or uploaded to the server

  4. Upload the converted file to CloudParty
